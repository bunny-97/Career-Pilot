# 🚀 CareerPilot AI
> **현직자 빅데이터와 RAG 기술을 결합한 데이터 기반 커리어 설계 플랫폼**

CareerPilot AI는 파편화된 정보 속에서 방황하는 구직자들에게 실질적인 **'성장 경로'**를 제시합니다. 단순한 정보 제공을 넘어, AI가 실제 데이터를 근거로 개인 맞춤형 커리어 로드맵을 설계하고 실행 솔루션까지 연결하는 커리어 네비게이션 서비스입니다.

---

## 📌 1. 사업 배경 및 필요성 (Business Background)

### 🔍 시장의 문제점: "정보는 많지만 내 길은 없다"
* **목표 설정의 혼란:** 신입 구직자의 **23.3%**가 '직무 설정 및 진로 결정'을 가장 큰 어려움으로 꼽으며, 이는 조기 퇴사율(**27.9%**)의 핵심 원인이 됩니다.
* **정보의 비대칭성:** 구직자의 **84.5%**가 필요한 정보를 찾지 못하거나 조언을 구할 인맥이 부족하여 구직 활동의 위축을 경험합니다.
* **기존 서비스의 한계:** 공고 중심의 단발성 플랫폼은 개인의 장기적인 성장 경로와 실행 계획을 지원하지 못하는 한계가 있습니다.

### 🎯 목표 시장 (Target Market)
* **초기 타겟:** 데이터 분석에 최적화된 **IT 분야 취업 준비생 및 주니어 이직자**.
* **확장 시장:** 연간 250만 명에 달하는 대학생 및 경력 전환 그룹을 포함하여 마케팅, 금융 등 **전문직 전체 시장**으로 확장.

## 🧠 2. 핵심 기술: RAG 기반 커리어 엔진 (The Core: RAG)

CareerPilot AI는 **RAG(Retrieval-Augmented Generation, 검색 증강 생성)** 방법론을 통해 LLM의 고질적인 문제인 '환각(Hallucination)'을 제거하고 높은 신뢰도를 확보했습니다.

### 🛠 RAG 시스템 작동 원리
1. **커리어 특화 DB 구축:** 실제 현직자의 상세 이력 및 최신 취업 정보를 벡터 데이터베이스화합니다.
2. **키워드 및 의도 추출:** `Kiwi` 형태소 분석기를 사용하여 사용자의 역량과 목표에서 핵심 키워드를 추출합니다.
3. **유사도 기반 데이터 검색:** 추출된 키워드를 임베딩하여 수백만 건의 데이터 중 가장 관련성이 높은 **'실제 성공 사례'**를 탐색합니다.
4. **근거 기반 로드맵 생성:** 검색된 데이터를 LLM(GPT-4 등)에 컨텍스트로 제공하여, **"누가, 어떤 과정을 거쳐 성공했는지"**에 기반한 초개인화 로드맵을 생성합니다.

### ✅ RAG 도입의 이점
* **신뢰성:** AI의 가상 답변이 아닌, DB 내 **실존 데이터**를 바탕으로 응답합니다.
* **최신성:** 모델 재학습 없이 DB 업데이트만으로 **최신 채용 트렌드**를 즉각 반영합니다.
* **정교함:** '백엔드'와 '서버 엔지니어' 같은 유의어를 의미 기반으로 정확히 매칭합니다.

## ✨ 3. 핵심 기능 (Key Features)

### 🗺️ 데이터 기반 인터랙티브 로드맵
* 수백만 명 현직자의 성공 데이터를 분석해 최적의 성장 경로를 **마인드맵** 형태로 시각화합니다.
* 사용자가 직접 노드를 수정하거나 AI와 대화하며 경로를 실시간으로 변경할 수 있습니다.
* *ex) "6개월 단기 집중 경로로 바꿔줘", "국내 활동 위주로 수정해줘"*

### 🚀 실전 실행 솔루션 연결 (Actionable Guide)
* 로드맵의 각 단계(노드)에서 필요한 역량을 강화할 수 있는 **온라인 강의, 사이드 프로젝트, 자격증** 과정으로 즉시 연결합니다.
* 단순 진단을 넘어 실제 '행동'으로 이어지는 가교 역할을 수행합니다.

### 🤝 롤모델 기반 소셜 검증
* 목표 직무에 도달한 실제 롤모델의 로드맵을 확인하고, 신뢰도 높은 유저들과 네트워킹할 수 있습니다.
* 인증 마크가 부여된 인재의 실제 경로를 통해 사회적 증거(Social Proof)를 제공합니다.

## 📈 4. 비즈니스 모델 및 성장 전략 (Business Strategy)

### 💰 수익 구조
* **B2C:** 고도화된 기능 및 상세 데이터를 제공하는 **프리미엄 구독 모델**.
* **B2B2C:** 제휴 교육 플랫폼(인프런, 패스트캠퍼스 등)과의 **강의 연결 수수료**.
* **B2B:** 대학교 취업지원센터 대상 **취업 관리 SaaS 솔루션** 공급.

### 🚀 데이터 전략: Give-to-Get
* 현직자에게 '커리어 프로필 빌더'와 '연봉 리포트'를 무료 제공하고, 서비스는 양질의 **현직자 경로 데이터**를 확보하여 강력한 기술적 해자(Moat)를 구축합니다.

### 🎯 성과 목표
* 1년 내 IT 분야 점유율 10% 확보.
* 3년 내 전문직 전 분야 확장 및 B2B SaaS 시장 안착.
